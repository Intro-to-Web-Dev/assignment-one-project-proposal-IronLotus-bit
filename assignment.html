Mission Statement:

For my Cweb class I will attempt to create a website to a game called Dungeons and Dragons. Dungeons and Dragons is a diffecult game to get into due to the game being very complex to new players. Unless the player is very dedicated or has the help of their host or friends, it is very easy for someone to be very put off about Dungeons and Dragons. That's why for my website the goals that I set for will be:

  1. Have it easy to use, as to not confuse new players
  2. Have it easy to understand, as to be quick as possible to get information
  3. Have it be visually intresting and simple, as to pleasing to the eye

Model:

With these goals the only thing left is to find a website to model my website off of. The website I chose for my primary model is Youtube. This is mostly due to the website already being established and understandable.
